布尔检索模型


为用户提供布尔检索服务，用户不必关注整个模型的具体实现，只需给定检索的语料库，通过构造并提交相应的布尔检索表达 式，指定结果的输出参数，即可获取在该范围内的检索结果，即用户需求的文档信息。
接口名：BooleanRetrieval

输入参数
参数名 类型 说明
docsPath String 语料库路径
expression String 检索表达式文本内容
stopWords HashSet<String> 停用词集合
docsNum int 输出文档数量的最大值，默认为 Integer.MAX_VALUE

输出参数
参数名 类型 说明
relatedDocs HashMap<Integer,String> 满足检索表达式的所有文档

接口名：VectorSpaceModel

输入参数
参数名 类型 说明
query String 查询语句

输出参数
参数名 类型 说明
createIndex String 文档库的倒排索引
countIdf String 文档库各个词的IDF
segmentQuery String 对查询语句进行中文分词的结果
removePunctuation String 对查询语句去标点的结果
removeStopWords String 对查询语句去停用词的结果
countTf String 查询语句各词的TF
outPutVector String 查询向量
outPutEveryDoc_Tf String 各个文档中各词的TF
outPutEveryDoc_Vector String 文档向量
countSimilarities String 查询与各个文档的相似度
sortSimilarities String 查询结果

分词API


接口名：LexicalAnalysis

1. 接口定义
功能名 接口原型 功能描述
中文分词 DivideWord divide (String sentence, int max) 分词接口，将等待分词的语句作为string形式输入，同时获取词典中词组的最大长度。该接口主要针对分词，将一个汉字序列切分成一个一个单独的词。

2. 接口描述
将一段中文语句（不大于200字）切分成一个一个单独的词，中间用“/”隔开字词输出字符串格式的结果。

3. 参数说明
3.1．输入参数（divide对象中属性）
参数名 是否必选 类型 说明
sentence 是 string 分词段落作为字符串输入
max 是 int 字典词组的最大长度

3.2. 返回值类型
类型 说明
string 分词后用斜杠“/”分割一个个词，再使用字符串类型输入。

参考示例
在“需要分词的段落”中输入要分词的中文段落（200字以内），点击“分词”按钮就可以在“分词结果的呈现”文本框内查看到分词结果。

4. 限制与约束
4.1 输入文本长度
分词文本限制在200字以内，超过字数无法输入，文本要UTF-8格式，格式错误的不报错，分析结果会不正确。

词性标注API


基于thulac工具包，提供中文词性标注的功能。
接口名：POSTagging
输入参数
参数名 类型 说明
text String 待标注词性的文本

输出参数
参数名 类型 说明
result String 经过词性标注的文本

词性标记集：

n/名词 np/人名 ns/地名 ni/机构名 nz/其它专名 m/数词 q/量词 mq/数量词 t/时间词 f/方位词 s/处所词 v/动词 a/形容词 d/副词 h/前接成分 k/后接成分 i/习语 j/简称 r/代词 c/连词 p/介词 u/助词 y/语气助词 e/叹词 o/拟声词 g/语素 w/标点 x/其它
调用示例：（python代码）>

                            
    import requests
    import json

    def postTest():
        url = "http://127.0.0.1:5000/POSTagging/"
        data = {"content": "“我最想感谢的人是我的导师付磊教授。在他的指导下，我发表8篇SCI论文，
                             总影响因子超过110，这是我给自己最好的毕业礼物。”“希望以后有很多钱，
                             给母校盖几栋楼。”“室友在外地实习，我们带着他的照片来参加典礼，没想
                             到这家伙一早赶回来了。”现场采访环节，毕业生们说着笑着哭着，话语里满
                             含眷恋与感恩。"}
        headers = {"Content-Type" : "application/json"}
        r = requests.post(url=url, data=json.dumps(data), headers=headers)
        print(r.text)
    if __name__ == "__main__":
        postTest()

命名实体识别API


提供命名实体识别功能。用户只需简单的调用相关API接口即可获取到所需结果。
接口名：NamedEntityRecognition
输入参数
参数名 类型 说明
text String 待分析的文本

输出参数
参数名 类型 说明
classes list
命名实体识别结果，其中list元素包含以下字段
word string 实体词
wlist list 每一类实体词的列表
cls string
人名 PER
地名 LOC
机构名 ORG
